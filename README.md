# HARNESS

Soul-driven, Obsidian-native agent harness. See [HARNESS_Research.md](HARNESS_Research.md) for the design.

To use Harness as pure prompt instructions with any agent or model, feed [AGENTS.md](AGENTS.md) as the system prompt — it tells the reading agent what to read, in what order, and how to honor scope tags and the memory contract. Claude Code picks up [CLAUDE.md](CLAUDE.md) automatically; every other agent should start with `AGENTS.md`.
